# game-ref-forge

RefForge, Chinese name `Ling Gan Duan Zao Tai`, is a personal-first game asset reference research desk.

It collects source links from game asset and game design sites, normalizes them into reference records, classifies them by asset type and creative use, and helps turn references into original-asset ideas without becoming a download mirror.

## Project Identity

- Repository name: `game-ref-forge`
- Product name: `RefForge`
- Chinese name: `灵感锻造台`
- Git remote: `https://github.com/immortalbeating2/game-ref-forge.git`
- Current phase: project documentation baseline before Sites implementation

## First-Version Direction

The first version is a private reference and research tool:

- Manual URL-first reference intake.
- Metadata preview before save.
- Unified reference cards with source, license, taxonomy, and inspiration fields.
- Private-by-default public-safety status.
- One `references` table for first-version storage.
- No file upload, asset hosting, bulk crawling, or public route in v1.
